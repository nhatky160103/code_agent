* The game is rendered in **true 3D** (tiles have thickness, blocks have height, real-time shadows).
* A **fixed isometric / angled top-down camera** is used (no free camera rotation).

## 2. Core Gameplay Objective

* Move the block across the map using valid rotations.
* Avoid falling off the map or triggering invalid tiles.
* Complete the level when the block:

  * Is **standing vertically**
  * Is positioned **exactly on the Goal tile**
